CP1406 - Web Design & Development - JCU

Practical - JavaScript

Note

This is not a walk-through style prac with step-by-step instructions.
Use the resources and sample code you've been given as references, and work with other students and your lecturer to achieve these tasks.

Aim:

  • To practise using JavaScript for client-side interactivity, focused on forms.

Tasks:

  1. Download the jsFormExamples.html and scripts.js files from https://github.com/lindsaymarkward/JavaScriptExamples and spend a minute or two trying out the functionality (read the notes at the bottom of the Web page).
  2. Edit the file and experiment with different techniques...
  3. Modify this form to achieve the following:
    1. Use the tick/cross image changing so that the required fields show crosses until they have valid entries in them (note that name might be valid but then the user could delete the name, so the tick should turn back to a cross).
    2. Notice how you can address any object and change its attributes, including CSS properties...
      Try setting some onfocus events for fields that don't do anything and set some CSS properties - like rounded corners for text fields, background-color for fieldsets... just to see how it works and practise it.
    3. You can see in a couple of places where the code gets the value of a text field...
      Make two new fields (don't have to be in this same form) that you can enter a width and height into;
      Use the values the user enters in here to set the width and height of an image on the page
      Also, write the total pixels (width * height) to the screen (using innerHTML)... this would be part of what you need to do for your shopping cart (total).
  4. Work on a form from your major project and add JavaScript form validation and interactivity as appropriate to your site requirements.
