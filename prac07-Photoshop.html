Practical - Photoshop

Note

This prac is not a step-by-step walk-through for learning Photoshop, but rather a set of tasks to accomplish.
Use the resources from SitePoint Premium, lynda.com or Adobe.com to learn to use the program:

  • Introduction to Photoshop (38 minutes)
  • Plus Adobe's tutorials

Aim:

  • To use Photoshop to achieve common Web-focused image editing outcomes, including:
    • Cropping
    • Resizing
    • Compressing
    • Basic bitmap editing
  • To create new (simple) images from scratch for things like logos, banners and buttons.

Tasks:

Download and unzip the original images provided to work on in this prac.

Create an empty folder called ps. For each task, export the resulting file to this folder.

Editing Images

  1. One of the most common imaging tasks you will have to do is resizing and compressing large images.
    1. Open the sunsetScenery.jpg image (a 4.5MB image).
    2. From the File menu, choose Export then Save for Web (Legacy) and change the top tab to 4-Up.
      This lets you experiment with different settings and see the expected results before you export the file.
    3. Click on one of the 4 panels and then change settings on the right.
      Click on a different panel and try other settings, comparing the results to the other panels.
    4. When comparing views of a large image like this, drag the image around in the view to show an area of high contrast to see the differences more clearly.
      For this photo, you will see the results better by looking at the power poles compared to looking at the clouds.
    5. Adjust settings to make the new file (always keep your original unchanged) as small as you can without obvious quality loss. How low can you go?
      Export an image file with the name, sunsetCompressed.jpg.
    6. Now resize the image from its massive 4288 pixel width down to 10% of its original size.
      Adjust settings, comparing some options, then export the best one as sunsetSmall.jpg.
    7. Do the same process of choosing a file format and compression settings for the file wordleDesignPrinciples.png.
      This file has very different image content compared to the sunset image...
      Do the same settings work well for each of them or are different settings better?
  2. Crop the dog image (dogOnBeach.jpg) so that you can't see the person's feet.
    Think about applying the rule of thirds to compose your new image.
  3. Lasso the toad off the couch (toadOnCouch.jpg) and place it on the grass next to the glove (gloveOnGrass.jpg).
    Think about scale so that it looks appropriate, and make sure you can't see any of the red and white couch on the grass.
  4. Someone has cruely* put ear rings on this cute baby (PCbaby.jpg). Edit the image to remove them so it looks like she has unpierced, normal ears. Try using the rubber stamp tool.
  5. acousticGuitars.jpg is a nice image... too nice. Use your creativity and any tools you like in Photoshop to create a different image that's still recognisable as an image of guitars.

Results Page

  • Create a new index.html page inside your ps folder.
  • For each of the tasks above, create a heading that briefly summarises the task, insert a thumbnail image that links to the full image (the result of the task, not the original source file).
    A thumbnail image is a smaller version of the image. It should be a new smaller image file, not just the same file with adjusted dimensions in HTML.
    You can create each thumbnail one at a time, or if you're feeling keen, use batch processing, as explained in this tutorial, which comes from one of our SitePoint books.
  • Save this page, then upload it with all of the dependent files (images and any CSS you used) to the server, so it's in a folder called ps.
    Check all of your links work.

Creating Images

    Start with a blank canvas in Photoshop.
    Use the drawing and text tools to come up with something...
    Export it and add it to your Training Zone site.
  2. Upload your newly image-upgraded Training Zone website to the server.
